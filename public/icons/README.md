# Quivo icons (temporary MVP)

PNG icons in this folder are **temporary** placeholders generated from brand colors.

- `icon-192.png` — standard launcher icon
- `icon-512.png` — splash / install icon
- `icon-maskable-512.png` — maskable safe-zone variant
- `icon.svg` — vector fallback for browsers

Regenerate PNGs after design changes:

```bash
npm run icons
```

Replace with final brand assets before production release.
